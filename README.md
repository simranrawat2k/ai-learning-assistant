# AI Learning Assistant

A RAG-based AI assistant that allows users to upload PDF files and ask questions about them.

## Features

- PDF Upload
- Semantic Search
- Chroma Vector Store
- Groq LLM
- Source References
- Chat History

## Tech Stack

- Python
- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Groq
